​This project is a desktop application developed using the Tkinter library in Python, designed to function as an interactive Blood Group Predictor. The application utilizes the established biological principles of ABO blood group inheritance and the Rh factor genetics. By allowing users to input the blood groups of both the mother and the father, the system dynamically processes the possible parental allele combinations. It then calculates and displays the scientific probability of the offspring's potential blood group.

This project is a desktop-based Purchase and Sales Inventory Management System built using Python's Tkinter library and powered by a backend database for persistent storage. The application provides a real-time, automated solution for retail stock tracking by dynamically calculating inventory balances based on transactional inputs. When a purchase is registered, the system commits the data to the database and automatically increments the remaining product stock. Conversely, when a sale is finalized, the system processes the deduction and instantly updates the interface to display the reduced product quantity remaining. This database-driven architecture eliminates manual ledger errors, ensures data integrity across sessions, and gives users a reliable, live view of their current inventory levels.
